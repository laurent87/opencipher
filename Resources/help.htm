About

Open Cipher

Open Cipher is a free encryption software based on OpenSSL. Developed using .NET and C# it is an interface between the user and OpenSSL security features. OpenSSL is a widely used and trusted open source security tool.

Usage

Encrypt current text selection

Using the hotkey Control + Alt + S will trigger the function to encrypt the current selected text. This feature is very similar to the Encrypt clipboard function as it is simulating a Copy and Paste of the data.

Encrypting file

There are two ways to encrypt or decrypt of file, you can use the Encrypt/Decrypt file menu entry then select the file you want to protect. Or using windows explorer right click on the file and select Encrypt or Decrypt menu entry.

Encrypting/Decrypting clipboard

It is possible to encrypt the clipboard content pressing the hotkey Control + Alt + E or using the "Encrypt clipboard" menu entry in the Open Cipher menu.

To decrypt the clipboard content press the hotkey Control + Alt + D or use the "Decrypt clipboard" menu entry in the Open Cipher menu.

Frequently asked questions

What is base 64 ?

Base 64 is an encoding method that converts any data into a string of readable characters. Without data loss it can be used for instance to send an image via a text file.
